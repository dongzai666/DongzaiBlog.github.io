# 前端三剑客之CSS
>将前端比作人体，那么CSS就是人体的肌肉

>CSS 是一种描述 HTML 文档样式的语言。CSS 描述应该如何显示 HTML 元素。

什么是 CSS？<br>
  * CSS 指的是层叠样式表* (Cascading Style Sheets)<br>
  * CSS 描述了如何在屏幕、纸张或其他媒体上显示 HTML 元素<br>
  * CSS 节省了大量工作。它可以同时控制多张网页的布局<br>
    外部样式表存储在 CSS 文件中
